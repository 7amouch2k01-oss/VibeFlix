# VibeFlix - Movie App

## Overview

VibeFlix is a dynamic web application that allows users to discover movies, view details, search by genre, and manage a personal watchlist. It leverages The Movie Database (TMDb) API to provide up-to-date information on trending movies, search results, and detailed movie profiles, including trailers and cast information.

## Features

*   **Movie Discovery**: Browse trending movies on the homepage.
*   **Search Functionality**: Search for movies by title.
*   **Genre Filtering**: Filter movies by various genres.
*   **Movie Details**: View comprehensive details for each movie, including synopsis, release date, rating, runtime, cast, and similar movies.
*   **Trailer Playback**: Watch official movie trailers directly within the app.
*   **Watchlist Management**: Add and remove movies from a personalized watchlist, persisted locally.
*   **Hero Section**: Features a prominent display of a currently playing movie with key information and quick actions.
*   **Responsive Design**: Optimized for seamless viewing across different devices.
*   **Loading Indicators & Error Handling**: Provides user feedback during data fetching and in case of errors.

## Technologies Used

*   **HTML5**: For structuring the web application.
*   **CSS3**: For styling, animations, and ensuring a responsive and visually appealing user interface.
*   **JavaScript**: Powers all interactive elements, API calls, and dynamic content rendering.
*   **The Movie Database (TMDb) API**: Serves as the primary data source for movie information.
*   **Local Storage API**: Used to store and retrieve the user's watchlist.

## Installation and Usage

To get VibeFlix up and running on your local machine:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/7amouch2k01-oss/VibeFlix.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd VibeFlix
    ```
3.  **Open `index.html`**: Simply open the `index.html` file in your preferred web browser. No additional server setup is required.

    **Note**: You will need to obtain your own API key from [TMDb](https://www.themoviedb.org/documentation/api) and replace `CONFIG.API_KEY` in `app.js` with your key for the application to function correctly.
